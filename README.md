# 🚗 Moto Company Data Analysis

## 📊 Project Overview
This project analyzes used car sales data to uncover key factors affecting car prices and market trends.

## 🔍 Key Insights
- Automatic cars dominate the market (>96%)
- Strong correlation between MMR and Selling Price (0.98)
- Mileage negatively impacts price (-0.58)
- Most cars fall in the mid-range (10K–15K)
